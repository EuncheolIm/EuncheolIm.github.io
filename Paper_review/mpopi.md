# Control of Legged Robots using Model Predictive Optimized Path Integral


### Intro
Current algorithms for legged locomotion and manipulation are generally divided into two main groups: 1. MPC, 2. RL
* MPC : do not require offline training, but it has computationally issue using in real-time
* RL : has shown impressive progress in recent years, but it remains a serious challenge, mainly due to the sparse rewards in complex environments
